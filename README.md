# ShareToCare 
A React + Tailwind CSS food donation platform that helps reduce food waste by connecting donors with people in need.

## Features

- Home Page
- Donate Food
- Available Food
- My Donations
- Search & Filter
- Contact Form
- Responsive Design
- 404 Page
- Loading Screen

## Tech Stack

- React.js
- Tailwind CSS
- React Router DOM
- Local Storage
